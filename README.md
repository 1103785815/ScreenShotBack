# ScreenShotBack
ScreenShotBack，截图手势返回，景深效果，类似斗鱼APP的手势返回。对于每个VC都可以关闭或者打开手势返回，一个属性搞定。

    youViewController.enablePanGesture = YES;（默认YES，支持手势返回）
    youViewController.enablePanGesture = NO;（NO，关闭手势返回）
